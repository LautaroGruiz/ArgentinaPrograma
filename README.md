El formulario funciona gracias a la herramienta <a href="https://formspree.io/">Formspree</a>. Al enviar un formulario se recibe los datos enviados al mail de un integrante del grupo (puede recibirse en cualquier mail pero debe contar con una cuenta en Formspree). Se requiere como obligatorio los datos de nombre, apellido e email.
